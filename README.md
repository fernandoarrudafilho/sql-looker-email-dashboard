# SQL + Looker Studio Email Dashboard

## Overview

This project combines Advanced SQL and Looker Studio to analyze email campaign performance, customer engagement, and country-level marketing metrics.

The dataset was transformed using SQL and visualized through an interactive dashboard for business decision-making.

---

## Tools Used

- SQL (Advanced)
- BigQuery
- Looker Studio
- Data Visualization
- Marketing Analytics
- KPI Reporting

---

## SQL Skills Demonstrated

- CTEs (Common Table Expressions)
- JOINs
- FULL OUTER JOIN
- Aggregations
- COALESCE
- DENSE_RANK()
- Filtering logic
- Business KPI calculations

---

## Dashboard Insights

- Email volume over time
- Open and visit rates
- Verified vs unverified users
- Top countries by emails sent
- Send interval performance
- Campaign engagement metrics
